## Finance Dashboard 

### Overview

This project is a responsive Finance Dashboard built using React and Tailwind CSS. It allows users to track, visualize, and analyze their financial transactions through an intuitive and user-friendly interface.

The dashboard includes role-based access (Viewer and Admin), interactive charts, filtering and sorting capabilities, and insightful analytics such as monthly comparisons and spending patterns.


### Features

* Transaction management (Add, Edit, Delete – Admin only)
* Search, filter, and sort transactions
* Data visualization using Recharts (bar, pie, and line charts)
* Financial insights (highest spending category, monthly comparison, daily averages)
* Responsive design for different screen sizes
* Dark mode support
* Clean and minimal UI inspired by modern design systems



### Tech Stack

* React (Frontend framework)
* Tailwind CSS (Styling)
* Recharts (Data visualization)
* Lucide React (Icons)


### Getting Started

To run this project locally, follow these steps:

1. Clone the repository


git clone https://github.com/SaumyaBelgi/Finance-Dashboard-UI.git


2. Navigate into the project folder


cd Finance-Dashboard-UI


3. Install dependencies


npm install


4. Start the development server


npm run dev


5. Open in browser
   The app will typically run at:


http://localhost:5173

### Notes

* All dependencies used in the project are listed in the `package.json` file.
* Running `npm install` will automatically install all required packages.
* No additional manual package installation is required.
